tBox - A slector plugin

Selctor plugin that converts you select, checkbox and radio buttons to selectable buttons

How to ?

Just simple, add the css and js to your project

1<link rel="stylesheet" type="text/css" href="tBox.min.css" />
2<script type="text/javascript" src="tBox.min.js"></script>

Note.: Please include jQuery before adding the tBox js file

Usage

  • Single Select
  • Multiple Select
  • Checkbox
  • Radio
1<select id="single-select" name="select-name">
2    <option value="value 1">Select Option 1</option>
3    <option value="value 2">Select Option 2</option>
4</select> 1$('#single-select').tBox();
